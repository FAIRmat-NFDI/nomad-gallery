The Alexandria Database hosted by the NOMAD Calculations App¶

The Alexandria database is being ingested into NOMAD! The result is tens of millions of NOMAD calculations entries that can be efficiently queried and explored via NOMAD's extensive search and filtering functionalities.

Search interface for the perovskite solar cell database¶

Field: Photovoltaics, Inst: Humboldt-Universität zu Berlin, (DE)

A search interface for exploring the perovskite solar cell database using materials, composition, and device-property filters to support comparison and analysis of photovoltaic performance.
